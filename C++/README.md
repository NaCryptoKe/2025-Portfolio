### March: Object-Oriented Programming (OOP) Basics  
**Focus:** Learn the fundamentals of classes, objects, constructors, and encapsulation.  

#### Key Topics:
- Access specifiers (`public`, `private`)  
- Method overloading  
- Debugging with `gdb`/IDE tools  

#### Projects:
1. BankAccount Class: deposit, withdrawal, balance checks  
2. Student Grade Tracker: multiple subjects, GPA calculator  
3. Library Management System: borrow/return tracking  

---

### April: STL Containers & Algorithms  
**Focus:** Master STL containers like `vector`, `map`, and practice sorting and searching algorithms.  

#### Key Topics:
- `std::vector`, `std::map`  
- Iterators and range-based loops  
- Sorting and searching (`std::sort`, `std::find`)  

#### Projects:
1. To-Do List Manager  
2. Word Frequency Counter  
3. Inventory Tracker  

---

### May: Memory Management & Smart Pointers  
**Focus:** Master RAII and smart pointers for safe memory management.  

#### Key Topics:
- `std::unique_ptr`, `std::shared_ptr`  
- RAII (Resource Acquisition Is Initialization)  

#### Projects:
1. Custom Dynamic Array (like `std::vector`)  
2. Resource Handler (e.g., managing file connections)  
3. Smart Pointer Practice (linked list using `shared_ptr`)
