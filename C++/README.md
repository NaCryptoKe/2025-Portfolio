# Comprehensive C++ Learning Roadmap

### **Goal**  
Build a strong foundation in C++ while progressively creating projects to reinforce and expand your understanding of key concepts.

---

## **Monthly Breakdown**

### **February: C++ Basics & Fundamentals**  
**Focus:** Master the basics of syntax, control flow, functions, and file I/O.  

#### Key Topics:  
- Variables and data types  
- Arithmetic and conditional operators  
- Control structures: loops (`for`, `while`)  
- Functions: parameters and return types  
- Arrays and basic file I/O (`fstream`)  
- Debugging basics (using IDE tools and `cout` debugging)  

#### Weekly Breakdown:
**Week 1: Variables, Operators, and Control Structures**  
- **Daily Projects:**  
  1. Unit Converter (e.g., miles â†” kilometers)  
  2. Number Guesser (using `rand()`)  
  3. Simple Calculator (using `switch` statements)  
  4. Multiplication Table Generator (using `for` loops)  
  5. Sum of Numbers Calculator (using `while` loops)  
  6. Prime Number Checker  

**Week 2: Functions and Arrays**  
- **Daily Projects:**  
  1. Function-Based Validation (`bool isValidNumber(string input)`)  
  2. Array Reversal  
  3. Average Score Calculator  
  4. Password Strength Checker  

**Week 3: File I/O Basics**  
- **Daily Projects:**  
  1. Write to a File (`ofstream`)  
  2. Read from a File (`ifstream`)  
  3. File Appender  
  4. Word Counter  

**Week 4: Final Projects**  
- **Portfolio Projects:**  
  1. Temperature Converter (Celsius â†” Fahrenheit)  
  2. Text File Analyzer (count lines, words, characters)  
  3. Add a Menu System (loop-based navigation)  

---

### **March: Object-Oriented Programming (OOP) Basics**  
**Focus:** Classes, objects, constructors, and encapsulation.  

#### Key Topics:  
- Access specifiers (`public`, `private`)  
- Method overloading  
- Basic debugging with `gdb` or IDE tools  

#### Projects:  
1. Bank Account Class (deposit, withdrawal, balance checks)  
2. Student Grade Tracker (grades for multiple subjects, GPA calculation)  
3. Library Management System (track borrow/return logic)  

---

### **April: STL Containers & Algorithms**  
**Focus:** Master `std::vector`, `std::map`, and algorithms.  

#### Key Topics:  
- Iterators and range-based loops  
- Sorting and searching with `std::sort` and `std::find`  

#### Projects:  
1. To-Do List Manager  
2. Word Frequency Counter  
3. Inventory Tracker  

---

### **May: Memory Management & Smart Pointers**  
**Focus:** Manual and automated memory management.  

#### Key Topics:  
- RAII (Resource Acquisition Is Initialization)  
- `std::unique_ptr`, `std::shared_ptr`  

#### Projects:  
1. Custom Dynamic Array (mimic `std::vector`)  
2. Resource Handler (`std::unique_ptr` for file/network connections)  
3. Smart Pointer Practice (linked list using `std::shared_ptr`)  

---

### **June: Advanced OOP & Templates**  
**Focus:** Inheritance, polymorphism, and generic programming.  

#### Key Topics:  
- Virtual functions and abstract classes  
- Template classes/functions  

#### Projects:  
1. Shape Hierarchy (area calculations for Circle, Rectangle, etc.)  
2. Generic Math Library (templated functions for common operations)  
3. Template Container (generic Stack or Queue)  

---

### **July: Graphics & Game Development Basics (SDL/SFML)**  
**Focus:** Rendering, input handling, and game loops.  

#### Key Topics:  
- Event-driven programming  
- Collision detection basics  

#### Projects:  
1. Pong Game (two-player paddle controls, ball physics)  
2. Interactive Paint Tool  
3. Simple Animation (sprite animation using spritesheets)  

---

### **August: Multithreading & Concurrency**  
**Focus:** Parallel execution and thread safety.  

#### Key Topics:  
- `std::thread`, `std::mutex`  
- Thread pools and task queues  

#### Projects:  
1. Multithreaded File Processor  
2. Threaded Task Scheduler  
3. Parallel Data Processor  

---

### **September: 3D Graphics with OpenGL**  
**Focus:** 3D rendering and transformations.  

#### Key Topics:  
- Vertex buffers, shaders (GLSL)  
- Model-view-projection matrices  

#### Projects:  
1. 3D Cube Renderer  
2. First-Person Camera  
3. Texture Loader  

---

### **October: Physics & Collision Detection**  
**Focus:** Basic physics simulations.  

#### Key Topics:  
- Gravity, velocity, and acceleration  
- AABB (Axis-Aligned Bounding Box) collisions  

#### Projects:  
1. Bouncing Ball Simulator  
2. Simple Platformer  
3. Collision Response  

---

### **November: Game Engine Framework**  
**Focus:** Core engine components.  

#### Key Topics:  
- Entity-Component-System (ECS)  
- Scene management and serialization  

#### Projects:  
1. 2D Game Engine  
2. Scene Manager  
3. Entity Component System  

---

### **December: Optimization & Data Structures**  
**Focus:** Performance tuning and algorithms.  

#### Key Topics:  
- Profiling with `gprof` or Valgrind  
- A* pathfinding and graph traversal  

#### Projects:  
1. Optimize Your Engine  
2. Pathfinding Visualizer  
3. Memory Pool Allocator  

---

### **January: Modern C++ & Final Project**  
**Focus:** Modern features and final consolidation.  

#### Key Topics:  
- C++17/20 features (`std::optional`, `std::variant`)  
- Refactoring and code quality  

#### Projects:  
1. Code Refactoring (modernize older projects)  
2. Final Game: Maze Runner  
3. Multi-Module Project  

---
